Scenario 1 # Having Repository in Github and wants to clone with Laptop local so that we can make changes and at loacl and push directory to git hub 
1. git clone <repository-link>   #### Repository link you can copy from github repository which you want to clone at local (Code sections copy https url)
2. git status
3. git add .
4. git commit -m "meaningful message for tracking the changes"
5. git push origin main

Scenario 2 # In case you are making local folder and want to push to Hithub
1. First make the loacl folder go to inside the folder from visual studio code terminal.
2. git init (from VS code Terminal)
3. go to github and make new repository there and open in hithub for copying the https url (<-link->)
4. git remote add origin <-link->
5. git remote -v (to verify the remote)
6. git branch (to verify the branch)
7. git branch -M main (to rename the branch to Main)
8. git push origin main
